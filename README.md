# SvelteKit + Windi CSS starter

This project uses only the minimized template of Sveltekit.

## Use this skeleton

```sh
degit thehelium/sveltekit-windi ${your new floder name or './'}

or

git clone https://github.com/thehelium/sveltekit-windi.git


pnpm  ## npm i or yarn

pnpm dev
```
